1. 코치의 이름 입력받기
- [x] 코치의 이름은 2글자 이상, 4글자 이하
- [x] 코치의 인원은 2명 이상, 5명 이하

2. 각 코치마다 못 먹는 메뉴 등록
- [x] 못먹는 메뉴는 0개 이상, 2개 이하

3. 메뉴 추천 기능
- [ ] x요일에 추천할 카테고리 정하기
- [ ] 같은 카테고리는 최대 2회까지 가능
- [ ] 해당 카테고리에서 각 코치가 x요일에 먹을 메뉴 추천
- [ ] 한명의 코치에게 중복된 메뉴 추천 x
- [ ] 먹지 못하는 메뉴는 추천 x
- [ ] 위 과정 월~금요일 5번 반복하여 각 코치별 추천메뉴 리스트에 등록

4. 추천 결과
- [ ] 월요일~금요일 출력
- [ ] 선택한 카테고리 출력
- [ ] 각 코치별 추천된 메뉴 출력

5. 에러처리
- [ ] 잘못된 값 입력시[ERROR] 에러 출력 후 그 부분부터 입력을 다시 받는다.
